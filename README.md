\# Quantum Computing Mini Projects



This repository contains beginner-friendly quantum computing simulations implemented using Qiskit.



\## Projects



\### 1. Quantum Coin Flip

Simulates a quantum coin using a Hadamard gate to demonstrate superposition.



\### 2. Entangled Coin Flip

Creates a Bell state using Hadamard and CNOT gates to demonstrate quantum entanglement.



\### 3. BB84 Quantum Key Distribution

Simulates the BB84 protocol for secure quantum communication.



\### 4. BB84 with Eavesdropping Attack

Models an intercept-resend attack by Eve and demonstrates error detection.



\### 5. Quantum Noise Simulation

Simulates depolarizing noise and compares ideal vs noisy quantum circuits.



\## Technologies Used

\- Python

\- Qiskit

\- NumPy

\- Matplotlib



\## Concepts Demonstrated

\- Quantum Superposition

\- Quantum Entanglement

\- Quantum Cryptography

\- Quantum Noise and Error Analysis
